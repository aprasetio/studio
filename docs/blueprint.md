# **App Name**: VersoKit

## Core Features:

- Dashboard Landing: Central landing page with navigation cards to available tools like Futsal Scoreboard and Lineup Builder.
- Futsal Scoreboard: Digital scoreboard featuring a 20-minute countdown timer, home/away score trackers, and foul counters.
- State Persistence: Utilize local storage to persist the state of the Futsal Scoreboard for uninterrupted use.
- Lineup Builder: A drag-and-drop interface to construct football lineups, rendered visually on a green pitch.
- Image Download: Enable users to download a visual representation of the created football lineup as an image.
- Internationalization (i18n): Support for multiple languages (English and Indonesian) using next-intl for a localized user experience.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A), conveying trust and reliability.
- Background color: Light gray (#F9FAFB), providing a clean, neutral backdrop.
- Accent color: Vibrant orange (#EA580C), drawing attention to interactive elements.
- Body and headline font: 'Inter' for a modern, neutral feel.
- Crisp, modern icons from a set like 'Remix Icon' or 'Tabler Icons'.
- Clean, card-based layout with Tailwind CSS spacing and sizing for optimal viewing on all devices.
- Subtle transition animations for timer start/pause and score updates for a smooth user experience.
