# VersoKit

This is a Next.js application built with Firebase Studio, featuring a collection of useful web tools.

## Features

- **Futsal Scoreboard**: A digital scoreboard for futsal games with a timer, score, and foul tracking. State is persisted in local storage.
- **Football Lineup Builder**: A tool to create and visualize football team formations. You can download the lineup as a PNG image.
- **Internationalization (i18n)**: Supports English and Indonesian.

## Getting Started

To get started, run the development server:

```bash
npm run dev
```

The application will be available at `http://localhost:9002`.
