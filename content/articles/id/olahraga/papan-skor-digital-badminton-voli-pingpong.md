Pertandingan badminton seru tapi semua orang ribut soal skornya karena tidak ada yang mencatat dengan benar. Wasit harus teriak-teriak angka dari pinggir lapangan. Penonton tidak tahu berapa skor tanpa bertanya. Situasi seperti ini bisa dihindari sepenuhnya dengan papan skor digital yang ditampilkan di layar — mudah dilihat semua orang, selalu akurat, dan tidak perlu diinstal.

Artikel ini menjelaskan cara menggunakan papan skor digital untuk Badminton, Bola Voli, dan Pingpong — termasuk panduan singkat sistem skor masing-masing olahraga agar tidak ada lagi perdebatan soal hitungan.

## Mengapa Papan Skor Digital Lebih Baik dari Manual?

Banyak komunitas olahraga masih mengandalkan papan skor manual (flip board), kertas, atau sekadar hitungan dalam kepala. Masalahnya:

- **Tidak bisa dilihat semua orang** — papan kecil di pinggir lapangan hanya terlihat dari dekat
- **Rawan kesalahan** — pencatat distraksi satu detik, skor sudah berubah
- **Tidak ada history** — kalau terjadi sengketa, tidak ada catatan bagaimana skor sampai di angka tersebut
- **Ribet di akhir set** — harus ingat siapa yang serve, siapa yang break, kapan setting dimulai

Papan skor digital di layar tablet atau laptop besar mengatasi semua masalah ini. Mudah dilihat dari kejauhan, selalu akurat, dan ada tombol undo jika ada salah input.

## Panduan Sistem Skor: Badminton

Badminton menggunakan sistem **Rally Point** sejak 2006 — setiap rally menghasilkan poin, tidak peduli siapa yang serve.

**Aturan dasar:**
- Best of 3 set (menang 2 set lebih dahulu)
- Setiap set dimainkan sampai **21 poin**
- Jika skor 20-20, siapa yang unggul 2 poin menang (deuce)
- Maksimum **30 poin** — siapa yang mencapai 30 menang meski selisih hanya 1 poin
- Jika ada set ke-3, pemain/tim yang mencapai **11 poin** pertama kali berhak memilih sisi lapangan

**Setting** (perpindahan serve):
- Serve berpindah setiap kali yang tidak serve mencetak poin
- Di set ke-3, sisi lapangan berpindah saat salah satu mencapai 11 poin

| Situasi Skor | Kondisi |
|---|---|
| 20-20 | Deuce — butuh unggul 2 poin |
| 29-29 | Sudden death — siapa yang poin berikutnya menang |
| 21 poin (tanpa deuce) | Set selesai |

## Panduan Sistem Skor: Bola Voli

Voli juga menggunakan **Rally Point System** — poin dicetak di setiap rally.

**Aturan dasar:**
- Best of 5 set (menang 3 set lebih dahulu) — untuk pertandingan resmi
- Untuk komunitas, umumnya **best of 3 set**
- Set 1–4: dimainkan sampai **25 poin** dengan selisih minimal 2 poin
- Set 5 (tie-break): dimainkan sampai **15 poin** dengan selisih minimal 2 poin
- Tidak ada batas maksimum poin — terus bermain sampai unggul 2 poin

**Rotasi:**
- Tim yang memenangkan rally dan mendapat serve, semua pemain rotasi searah jarum jam satu posisi
- Ini penting untuk diingat agar posisi pemain tidak salah

| Set | Target Poin | Set ke-5 |
|---|---|---|
| Set 1–4 | 25 poin | — |
| Set 5 | — | 15 poin |
| Deuce | Selisih 2 poin | Selisih 2 poin |

## Panduan Sistem Skor: Tenis Meja (Pingpong)

Pingpong modern menggunakan sistem yang lebih sederhana dibanding badminton dan voli.

**Aturan dasar:**
- Best of 5 atau best of 7 game (tergantung turnamen/kesepakatan)
- Setiap game dimainkan sampai **11 poin**
- Jika skor 10-10, deuce — butuh unggul 2 poin
- Tidak ada batas maksimum
- Serve berganti setiap **2 poin** (dulu 5 poin, diubah sejak 2002)
- Saat deuce, serve berganti setiap poin

| Format | Kemenangan |
|---|---|
| Best of 5 | Menang 3 game |
| Best of 7 | Menang 4 game |
| Target per game | 11 poin |
| Deuce | Selisih 2 poin, tanpa batas maksimum |

## Penggunaan Praktis: Tips per Konteks

### Untuk Komunitas GOR
- Gunakan tablet ukuran besar (10 inci ke atas) yang ditempatkan di posisi yang bisa dilihat dari kedua sisi lapangan
- Aktifkan mode layar selalu nyala agar tidak sleep di tengah pertandingan
- Tunjuk satu orang sebagai scorekeeper — biasanya pemain yang sedang istirahat di luar lapangan

### Untuk Turnamen Kecil (8–16 peserta)
- Gunakan 1 device per court sebagai scoreboard
- Minta wasit atau penonton netral yang mengoperasikan
- Screenshot skor akhir sebelum reset sebagai dokumentasi hasil pertandingan

### Untuk Latihan Rutin
- Papan skor digital membuat sesi latihan terasa lebih "serius" dan memotivasi pemain
- Catat pemenang setiap game untuk tracking progress mingguan

### Untuk Tayangan Langsung (Live)
- Hubungkan laptop ke TV besar via HDMI — papan skor bisa ditampilkan untuk penonton di tribun atau ruang tunggu
- Ini terutama berguna untuk turnamen komunitas yang ingin memberi pengalaman menonton yang lebih baik

## FAQ: Papan Skor Digital

**Q: Apakah tool ini bisa digunakan saat tidak ada koneksi internet di GOR?**

A: Ya. Setelah halaman tool dimuat di browser, semua fungsi bekerja secara offline. Data tidak perlu dikirim ke server — seluruh logika skor berjalan di browser. Pastikan halaman sudah terbuka dan terload penuh sebelum masuk area tanpa sinyal.

**Q: Bagaimana jika ada salah input poin?**

A: Ada tombol Undo di tool untuk membatalkan input terakhir. Ini mengatasi masalah umum "tidak sengaja tap dua kali" atau input poin ke tim yang salah. Tombol undo bisa diklik berulang kali untuk membatalkan beberapa input sekaligus.

**Q: Apakah bisa custom aturan skor untuk olahraga lain yang tidak ada di daftar?**

A: Tool Universal Scoreboard saat ini dioptimalkan untuk Badminton, Voli, dan Pingpong dengan aturan standar internasional. Untuk olahraga dengan aturan non-standar atau olahraga lain, gunakan mode "custom" jika tersedia, atau gunakan tool sebagai counter poin sederhana.

**Q: Berapa ukuran layar minimum yang direkomendasikan?**

A: Untuk penggunaan personal oleh wasit: HP 5–6 inci sudah cukup. Untuk ditampilkan ke pemain dan penonton: tablet 10 inci atau laptop. Untuk turnamen dengan penonton: sambungkan ke TV atau monitor besar via HDMI untuk visibilitas terbaik.
